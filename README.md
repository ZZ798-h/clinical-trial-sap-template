# Clinical Trial Statistical Analysis Plan Template

## Overview

This repository provides a reusable Statistical Analysis Plan (SAP) template for clinical trial and public health research projects.

The goal of this project is to demonstrate foundational knowledge of clinical trial biostatistics, statistical analysis planning, study endpoints, analysis populations, missing data handling, safety analysis, and reproducible reporting workflows.
